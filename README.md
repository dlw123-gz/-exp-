
# 基于XGBoost的股票选股模型项目介绍
## 一、项目简介
本项目是一套基于XGBoost机器学习算法的智能股票选股模型。相比于传统人工选股、公式选股，本模型通过机器学习自动学习历史股票数据规律，智能筛选出上涨概率更高、收益更好的个股，帮助用户降低选股难度、减少主观判断误差。
## 二、项目核心思路
股票涨跌受行情、资金、基本面、技术走势等大量因素影响，传统公式很难精准判断。XGBoost擅长处理复杂数据和非线性规律，可以自动分析海量历史数据，总结出适合A股的选股逻辑，从而实现智能化选股。
## 三、使用教程
本项目依赖通达信官方给出的股票数据进行模型训练与预测，可以个人接入接口也可以直接下载文件用当日处理和当日添加两个py脚本来更新数据，然后运行predictv37.py来进行预测
